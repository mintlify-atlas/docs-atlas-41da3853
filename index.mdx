## What is Metrics?

Metrics fetches data from GitHub's APIs and third-party services, then renders it as a beautiful SVG image you can embed anywhere. Use it as a GitHub Action for fully automated updates, or run your own web instance.

## Get started

<Steps>
  <Step title="Create a GitHub Personal Access Token">
    Go to [GitHub Settings → Developer settings → Personal access tokens](https://github.com/settings/tokens) and create a token with the `public_repo` scope (add more scopes for private data).
  </Step>
  <Step title="Add the token as a repository secret">
    In your GitHub profile repository (e.g. `username/username`), go to **Settings → Secrets and variables → Actions** and add a secret named `METRICS_TOKEN`.
  </Step>
  <Step title="Create a workflow file">
    Add `.github/workflows/metrics.yml` to your profile repository with your chosen plugins and options.
  </Step>
  <Step title="Embed in your README">
    Reference the generated image file in your `README.md` and watch your profile come to life.
  </Step>
</Steps>
